List is the component showing all under contents. It can be used for both location content(eg. article) and non-location content(eg. image)

A typical List use includes `columns`, `sort`, `sort_default`, `pagination`.

A list can have actions above the records(see `actions` property), or in row(see `row_actions` property). For more about actions see [Config actions](./actions/Config), [Implement an action](./actions/Implement)

## Usage
```js
//Show article under 10, which "name", "published", "modified" columns
const columns = ["name", "published", "modified"];
const sortDefault = ["priority desc", "modified desc"]
<List id={10} contenttype="article" columns={columns} sort_default={sortDefault} />

//Action with link
const articleActions = [{"link":"/main/edit/{id}", "name":"edit", icon:"fas fa-edit"}];
<List id={10} contenttype="article" columns={columns} sort_default={sortDefault} row_actions={articleActions} />

//Action with action component
const articleActions = [{"com":<Copy />, "name":"Copy", "icon":"fas fa-copy"}];
const articleActions = [<Action icon="fa fa-edit" name="Copy" component={Copy}];
const articleActions = [{"com":(actionProps:ActionProps)=><Copy {...actionProps} />, "name":"Copy", "icon":"fas fa-copy"}];
<List id={10} contenttype="article" columns={columns} sort_default={sortDefault} row_actions={articleActions} />


```
